# B2B Admin Workflow Guide

## Overview
This system allows the Admin to manage B2B Agencies as partners and create white-labeled itinerary proposals for them.

## 1. Manage Agencies
**Agencies** are business profiles. They do NOT log in. You manage everything for them.

-   **Go to**: `Admin Panel` -> `Agencies (B2B)`
-   **Add Agency**:
    -   Enter Company Name (e.g., "Bali Travels")
    -   Enter Contact Person & WhatsApp
    -   Select "Specializations" (Places they focus on)
    -   Set default markup %

## 2. Create a Proposal
When an agency asks for a quote or you want to send them a package:

-   **Go to**: `Admin Panel` -> `Itineraries` -> `B2B Itineraries` (create new)
-   **Step 1**: Select the Agency & Destination
    -   e.g., Agency: "Bali Travels", Place: "Bali", Duration: 5 Days
-   **Step 2**: Build Itinerary
    -   Add Day-by-Day details (Places, Hotels, Activities)
    -   Set Base Costs (Your cost)
    -   Set Markup % (Agency's profit or your B2B markup)
-   **Step 3**: Save & Share
    -   Click **"Share PDF"** to get a WhatsApp link or Download PDF.
    -   The PDF is branded with **"Bali Travels"** (The Agency), not your company.

## 3. Pricing Logic
-   **Base Cost**: The cost of hotels, transport, tickets entered in the builder.
-   **Markup**: Percentage added on top.
-   **Total Price**: What the end client sees on the PDF.

## 4. WhatsApp Sharing
The "Share PDF" button generates a link like:
`https://wa.me/?text=Check out this proposal... [PDF_LINK]`
Send this to the Agency contact!
