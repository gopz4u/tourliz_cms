QUICK UPLOAD COMMAND (Using ZIP):
===========================================
1. Create ZIP excluding:
   - .env
   - .git/
   - node_modules/
   - storage/logs/*.log
   - storage/framework/cache/data/*
   - storage/framework/sessions/*
   - storage/framework/views/*

2. Upload ZIP to server

3. Extract in web root

4. Run setup commands
